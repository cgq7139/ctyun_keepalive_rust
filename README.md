# 天翼云桌面保活工具 (Rust 版本)

这是一个用 Rust 编写的天翼云桌面保活工具，用于保持云桌面连接活跃，防止自动断开。

## 功能特性

- 自动登录天翼云桌面
- 支持多账号与多云桌面同时保活
- 验证码自动识别
- 配置文件存储（账号、密码与设备码加密）

## 使用方法

### 首次运行

```bash
cargo run --release
```

首次运行时会提示输入账号和密码，配置会自动保存到 `config.json` 文件中。
程序会自动读取配置文件并登录。

## 配置文件

配置文件 `config.json` 存储在项目根目录，包含以下信息（支持多个账号）：

- `accounts`: 账号列表
- `user_account`: 天翼云账号
- `password`: 密码（加密存储）
- `device_code`: 设备代码（自动生成并加密存储）

## 编译

```bash
cargo build --release
```

编译后的可执行文件位于 `target/release/ctyun_keepalive`

## 依赖

- Rust 1.93.1 或更高版本
- 互联网连接

## 注意事项
- 软件的配置文件已经做了相应的加密存储处理，但是请仍妥善保管配置文件
- 账号、密码与设备码以加密形式存储
- 建议在安全的环境中使用
